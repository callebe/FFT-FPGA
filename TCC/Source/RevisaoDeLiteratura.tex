
Um sinal é um conjunto de dados ou informações, que podem descrever diversos tipos de fenômenos, como um sinal de telefonia ou de voz (Figura \ref{fig:Voice}), o registro de vendas de uma empresa ou ainda os valores de fechamento de uma bolsa de valores (Figura \ref{fig:CotacaoBovespa}) \cite[p.~75]{Lathi}. Segundo \citeonline[p.~1]{Oppenheim} existe uma linguagem adequada para descrever sinais e um conjunto poderoso de ferramentas para analisá-los, capaz de se aplicar a problemas oriundos de diversos domínios. A série de Fourier e a Transformada Rápida de Fourier são ambas exemplos destas ferramentas, e serão apresentadas nos próximos capítulos.
